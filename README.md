# 사랑의 교실 결과통보서 도우미

아이가 손으로 작성한 **‘나?’ 자기이해 활동지** 사진을 올리면,
Gemini Vision이 손글씨를 읽어 결과통보서의 **종합의견 초안**을 만들어 주는 도구입니다.

> AI가 만든 것은 **초안**입니다. 사실관계와 표현을 상담사가 반드시 확인·수정한 뒤 사용하세요.

## 사용 방법
1. 하단 ⚙️ 설정에서 **Gemini API 키**를 입력합니다. (무료: https://aistudio.google.com/app/apikey)
2. **활동지 사진**을 올립니다. (여러 장 가능 · 밝은 곳에서 선명하게)
3. 이름·회기·일자와 *상담 중 관찰한 점*(선택)을 적습니다.
4. **종합의견 만들기**를 누릅니다.
5. AI가 읽은 답을 확인하고, 참여태도·종합의견을 수정한 뒤 **복사**해서 사용합니다.

API 키와 입력한 내용은 **사용하는 기기(브라우저)에만** 저장되며 별도 서버로 전송되지 않습니다.
(사진과 프롬프트는 종합의견 생성을 위해 Google Gemini API로 전송됩니다.)

## 자료 보완
활동지 문항이나 예시 의견을 추가·수정하려면 `src/App.jsx` 상단의
`PAGE1_PROMPTS`·`PAGE2_PROMPTS`(문항)와 `EXAMPLES`(예시 의견)만 고치면 됩니다.
그림 장(자아상 그리기)도 함께 올리면 그림 내용까지 해석에 반영됩니다.

## 개발
```bash
npm install
npm run dev      # 개발 서버
npm run build    # 배포 빌드 (dist/)
```
React + Vite, Vercel 정적 배포.
